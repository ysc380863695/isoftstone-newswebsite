## 问题汇总

### 严重（必须修复）
无

### 一般（建议修复）
1. **静态资源 404**
   - 位置：首页加载时
   - 控制台：`Failed to load resource: the server responded with a status of 404 (Not Found)` ×3
   - 建议：检查 favicon.ico 或其他引用的静态资源路径是否正确

### 建议（可选优化）
1. **Babel 浏览器编译警告**：生产环境建议预编译 JSX，去除 Babel standalone 依赖，减小首屏加载体积

聊天问答系统核心功能全部通过测试。WebSocket 连接稳定，Tavily 搜索和 LLM 流式回复链路完整，非软通问题的预过滤拒绝机制快速准确（纯本地判定，不消耗 API 调用）。前端 UI 状态管理正确——思考动画、输入框禁用/恢复、消息气泡渲染、来源引用链接均无异常。整体质量可上线。
