## 审查检查清单

### CRITICAL
- [ ] 硬编码密钥（API key、密码、token）
- [ ] SQL 注入
- [ ] XSS（未转义用户输入）
- [ ] 路径穿越
- [ ] CSRF、认证绕过
- [ ] 缺少输入校验

### HIGH
- [ ] 大函数（>50行）、大文件（>800行）
- [ ] 深层嵌套（>4层）
- [ ] 缺少错误处理
- [ ] console.log 残留
- [ ] 新代码缺少测试
- [ ] Doc-Code 一致性

### MEDIUM
- [ ] 低效算法
- [ ] React 不必要重渲染
- [ ] 缺少缓存
- [ ] N+1 查询
- [ ] Bundle 过大

### 项目维度
- [ ] RD-1 视觉品质（Pentagram杂志风）
- [ ] RD-2 数据管道可靠性（爬取→AI→存储）
- [ ] RD-3 资源效率（内存<3.2G, API<3s, SQLite<100MB）
- [ ] RD-4 API一致性（字段名与 api-contracts.md 一致）
